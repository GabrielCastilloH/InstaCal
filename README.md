# InstaCal

A Chrome extension that uses AI to create Google Calendar events from natural language input.

Simply type something like **"Lunch with Sarah tomorrow at noon"** and InstaCal will parse it and add it to your Google Calendar — no forms, no clicking through menus.

## Features

- Natural language event creation
- Google Calendar integration
- AI-powered date, time, and event detail parsing

## Tech Stack

- **Frontend**: React + TypeScript
- **Build Tool**: Vite
- **Platform**: Chrome Extension (Manifest V3)
